Users API

GET

List Users

/users

Returns a paginated list of all users.

Parameters

Name Type Required Description
page integer No Page number for pagination. Default is 1.
limit integer No Number of items per page. Default is 20, maximum is 100.
sort string No Field to sort by. Prefix with '-' for descending order. Example: 'name' or '-created_at'

Headers

Name Required Description
Authorization Yes Bearer token for authentication. Format: 'Bearer {token}'
Accept No Media type. Default is 'application/json'
