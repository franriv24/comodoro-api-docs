Welcome to the official documentation for the Comodoro Delivery REST API. This documentation will help you integrate with our services and build amazing applications.

Quick Start

Base URL

https://api.comodorodelivery.app.br

Authentication

All API requests require authentication using a Bearer token:

Authorization: Bearer YOUR_API_TOKEN
Learn more about authentication →

View User APIs →

Example Request

  
fetch('https://api.comodorodelivery.app.br/users', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  }
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));


  curl -X GET "https://api.comodorodelivery.app.br/users" -H "Authorization: Bearer YOUR_API_TOKEN" -H "Content-Type: application/json"
